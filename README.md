# 근무·급여 오차 추적기 (work-pay-tracker)

예정 근무 → 실제 근무 → 실제 입금액을 비교해 **어디서 차이가 났는지 확인하는 개인용 기록 도구**입니다.
급여 계산 앱이 아니며, 임금체불 여부나 법정 지급액을 판정하지 않습니다.

## 무엇을 하는가
- 근무지 1개와 기본 시급 설정
- 예정 근무 / 실제 근무 입력 (아는 값만, 빈 값은 "미입력")
- 시급 기준 단순 예상액 표시 (참고값)
- 급여기간 단위로 실제 입금액 입력 → 총차이(입금액 − 단순 총예상액) 확인
- 로그인 없이 기기 로컬 저장, 사용자가 직접 실행할 때만 내보내기

## 무엇을 하지 않는가 (동결 경계)
- 실제 입금액을 날짜별로 나누지 않음
- 특정 날짜를 "미지급/체불"로 부르지 않음
- 주휴·야간·연장·휴일수당·세금·공제 계산 없음
- 최저임금 위반·임금체불 판정 없음
- 미입력 값 자동 추정 없음

## 실행 (개발/검증)
```
# 자동 단위 테스트 (계산 계약 03 검증)
npm test          # = node --test

# 로컬 웹 서버로 열기 (file:// 아님 — ESM 로드를 위해 http 필요)
npm run serve     # http://localhost:8080  (app/ 를 서빙)
```

## 구조
```
app/            정적 웹 루트(배포 대상)
  index.html
  styles.css
  src/domain.js     순수 계산(03 도메인 계약)
  src/storage.js    기기 로컬 저장/내보내기
  src/analytics.js  로컬 카운터 이벤트(민감 원문 없음)
  src/legal.js      법률 경계 문구 상수(원문 동결)
  src/app.js        화면 조립
test/
  domain.test.mjs   계산 계약 자동 테스트
```

## 개인정보 · 법률 경계
표시되는 금액은 사용자가 입력한 근무시간과 기본 시급을 단순 계산한 참고값입니다.
주휴·야간·연장·휴일수당, 세금 및 기타 조건이 반영되지 않을 수 있으며
법률상 지급액이나 임금체불 여부를 판정하지 않습니다.

기록은 이 기기에만 저장되며 서버로 전송되지 않습니다. 백업은 사용자의 내보내기입니다.
